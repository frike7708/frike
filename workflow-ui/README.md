# 后台管理系统

> 基于Vue.js的前后端分离单页应用  

### 构建步骤:

``` bash
# 安装node依赖
npm install

# 服务热加载/刷新 @ localhost:8080
npm run dev

# 生产环境编译/打包/压缩
npm run build

# 生产环境打包编译并查看打包分析报告
npm run build --report
```
